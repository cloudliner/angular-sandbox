## Instructions

### Run

```sh
git clone git@github.com:cloudliner/angular-sandbox.git
npm install
ng serve
open http://localhost:4200/
```

### Debug

1. Install Visual Studio Code
2. Add Debugger for Chrome to Visual Studio Code

### Test

#### Running unit tests

```sh
ng test
```

#### Running end-to-end tests

```sh
ng e2e
```
